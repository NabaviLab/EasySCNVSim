Super Easy Simulation of Genetic Variations

Don't spend time learning command-line tools to generate data for your research studies.
VarSimLab makes it extremely easy to simulate somatic CNVs and structural variations (SVs) in your reference genome and generates synthetic FASTQ reads for you.

Normal Reads

First, you will generate reads for the normal genome.
They represent reads from a healthy person;
with normal SNPs and Indels.

Add Variations

Second, you will design your experiment to simulate the tumor genome.Select parameters the way you want them; No restrictions.

Tumor Reads

Third, you will generate reads for the tumor genome.
You will also get proper variation files included that are the truth benchmark.

2. MODIFY PARAMETERS

Now, modify the simulation parameters.
Normal reads will be generated from the original reference genome.
Tumor reads will be generated with the below simulation parameters.

SNP rate (%)
INDEL rate (%)
Transition/Transversion ratio


CNV regions
CNV min size (bp)
CNV max size (bp)


Ploidy
Number of Reads
Read Length (bp)


Number of Subclones




Output Prefix
